Clients

Current and Most recent projects
Trimble, Plano, TX

Trimble has an existing application designed to serve the Direct Store Delivery Industry. Affilia is developing the new generation of the project utilizing Microsoft .Net Compact Framework. Affilia staff in both Dallas and Kiev offices work closely with the client's developers.
www.trimble.com
Exceleron Software, Dallas, TX

Exceleron Software has an existing web application written in Cold Fusion that provides customer management, billing and ordering for telecommunication companies. Affilia is working with Exceleron on the new release of the application utilizing Microsoft .Net and ASP.NET. In addition, Affilia provides 24/7 application maintenance and monitoring.
www.exceleron.com
Caremark, Irving, TX

Caremark needed a solution enabling them to submit employee insurance enrollment information directly by small to midsize companies. Affilia designed an application utilizing C++ and Win32 API that allows small companies to enter enrollment information, validate it and submit it directly to the Caremark systems. Presently, the application is deployed to hundreds of Caremark clients.
www.caremark.com

Other selected projects

Affilia developed a solution for Exceleron Software that enabled its product to exchange EDI data with an ILEC (Incumbent Local Exchange Carriers) on behalf of their CLEC (Competitive Local Exchange Carriers) customers over public lines (Internet) providing security, authentication, and non repudiation.

Affilia worked with Sage Telecom (www.sagetelecom.net) to develop an Intranet/Internet solution that allowed their telemarketing department to retrieve data about company services while selling offering them to the customer. The same solution allowed customers to view available services and sign up for them directly via Internet.

Affilia created comprehensive sweepstakes functionality for the HIT Entertainment's www.barney.com web site that provided web based tools to manage the web site's data as well as select the sweepstakes winners for multiple events such as concerts, shows and monthly prizes.

Hobson Preview Details

This preview application gives the first approach of the way how to automate service works. This one is especially designed for Hobson™ Air Conditioning Inc.

  


The left screen shows the main menu which contains essential operations employee uses during the work day. The work day begins with pressing "Start Day" button which shows "Start Day" dialog. User enters the location that he is going to service this day, selects his name for identification and enters current date and time.

    

 


After "Start Day" operation completion the application changes its state to "Service" and allows user  to service customers or complete work day. Pressing "Customer Ops" shows the list of customers to be serviced and their state. Context menu allows you to add new customer to the service queue, skip it on account of something or make a phone call. Pressing "Select" menu item shows the "Customer Menu".

The "Customer Menu" contains "Customer Props" (customer personal information), "System Props" (information about installed system) and "Message" (information about work to be done).

  

  

  

Pressing "Next" button moves user to the "Service" part.

  

Here the employee selects the items group and sets the quantities next to some items. The next two screens shows "Receipt" and "Payment" dialogs.

  

Pressing "Finish" completes this customer servicing and brings user to the customer list to select another customer to service.

After servicing all the customers during the day user presses "End Day" button on main menu and reconciles his balance.

Pressing "Transfer" button from the main menu emulates data transfer process to the server system and get application ready for new work day.

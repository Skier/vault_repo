Q-Agent

Overview

  • Provides wireless access to QuickBooks Mobile Edition
  • Runs on Windows Mobile 5.0, Windows Mobile 2003
  • Optimized to be used without stylus if device has attached QUERTY keyboard
  • No software needs to be installed on the PC
  • Connects directly from Pocket PC to QuickBooks Online Edition
  

List of functionality

  • Time Tracking
    • Single activity time sheet
    • Weekly time sheet
  • Customer management
    • Quick customer search
    • Create and edit customers
    • Manage invoices
  • Banking
    • Write checks
    • Enter credit card charges
  • Accounts
  • View Employees, Vendors and Items
  • Quick synchronization with QuickBooks Online Edition
   

Time Tracking

Create time sheets and synchronize them with Quick Books Online Edition. Quickly create a time entry by using Single Time Sheet. You can review and edit time entries by using Weekly Time Sheet. Record your time by specifying customer, service provided, and time spent.

      


Weekly time sheet allows you to review all time entries for the week. New time entries can be created or existing time entries edited. If time entry is synchronized with the Quick Books Online Edition, it will not be editable.


If you want to create a new time entry based on an existing one, please use the "Copy" function to save time.

Customer management

Review your customer information and their current balance. Q-Agent allows you to quickly call you customer phone number.

        


Left column of the customer list shows the status of customer record. "C" - Created, "S" - Synchronized and "M" - Modified. Editing synchronized customer changes its status to "Modified". After next synchronization session all the changes will be uploaded to your Quick Books Online account and their status will become synchronized.


Create new and review existing invoices for each customer.

Quick Synchronization with Quick Books Online Edition

Q-Agent synchronized data directly with Quick Books Online Edition. All the changes that are made will be sent to your account and will be accessible when you login to your account using a web browser. All data is sent using secure 128 bit SSL protocol. To enable faster synchronization, you can restrict the transactions you need on your mobile device by time period.
